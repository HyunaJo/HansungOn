# HansungOn
2023 4-1 iOS 프로그래밍 프로젝트

## 프로젝트 소개
> 영화 소개 iOS 애플리케이션

OTT (Over-the-top) 시장이 발달함에 따라 OTT 및 영화 콘텐츠를 소개하는 서비스들의 중요성이 커지고 있다. 이에, 네이버 시리즈온 어플리케이션을 유사한 기능을 갖는 영어 데이터 기반의 OTT 소개 애플리케이션을 제작하였다.
